Installation

To install it easily, please run:

curl -sS https://raw.githubusercontent.com/idvoretskyi/vim_idv/master/install.sh | bash

Updating existing installation

Simply run the following command:

curl -sS https://raw.githubusercontent.com/idvoretskyi/vim_idv/master/update.sh | bash

Plugin Management

The full list of plugins one may find in '~/.vim/plugins.vim' file. To add any new plugin, simply add it to that file.

Example: I'd like to add the NERD tree plugin. I have to simply run:

echo "Plugin 'scrooloose/nerdtree' >> ~/.vim/plugins.vim && vim +PluginInstall +qall.

Detailed information about Plugin installation one may find in Vundle.vim documentation.

Bundled plugins

In my vim distribution the following Plugins are used:

  • christoomey/vim-tmux-navigator: Seamless navigation between tmux panes and vim splits

  • JarrodCTaylor/vim-shell-executor: Execute vim buffer in the shell and view output in split pane

  • andviro/flake8-vim: Flake8 plugin for Vim

  • scrooloose/nerdtree: A tree explorer plugin for vim.

  • klen/python-mode: Vim python-mode. PyLint, Rope, Pydoc, breakpoints from box.

  • davidhalter/jedi-vim: Using the jedi autocompletion library for VIM.

  • vim-scripts/nginx.vim: Highlights configuration files for nginx

  • tpope/vim-eunuch: helpers for UNIX

  • tpope/vim-fugitive: a Git wrapper

  • vim-scripts/conque-term: Run interactive commands inside a Vim buffer

  • jistr/vim-nerdtree-tabs: NERDTree and tabs together in Vim, painlessly

  • scrooloose/syntastic: Syntax checking hacks for vim

  • majutsushi/tagbar: Vim plugin that displays tags in a window, ordered by scope

  • scrooloose/nerdcommenter: Vim plugin for intensely orgasmic commenting

  • bling/vim-airline: lean & mean status/tabline for vim that's light as air

  • Xuyuanp/nerdtree-git-plugin: A plugin of NERDTree showing git status

  • rodjek/vim-puppet: Puppet niceties for your Vim setup

  • vim-ruby/vim-ruby: Vim/Ruby Configuration Files

  • fatih/vim-go: Go development plugin for Vim

  • vim-jp/vim-go-extra: Extra plugin for golang

  • rjohnsondev/vim-compiler-go: Vim compiler plugin for Go (golang)
